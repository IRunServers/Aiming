# BeizerManger.ModifyEndPoint
Modifies the end point

!!!note
    This is used to convert a `Vector3` end point from `Camera` mode to a `Vector2`

## Parameters
| Name       | Description | Type                | Default |
| ---------- | ----------- | ------------------- | ------- |
| `EndPoint` | N/A         | `Vector2 | Vector3` | N/A     |

## Return
`<Vector2>`
