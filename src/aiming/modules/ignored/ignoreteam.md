# Ignored.IgnoreTeam
Ignores a team from the `GetClosest` functions and `Aiming.Selected`

## Parameters
| Name        | Description | Type     | Default |
| ----------- | ----------- | -------- | ------- |
| `Team`      | N/A         | `Team`   | N/A     |
| `TeamColor` | N/A         | `Color3` | N/A     |

## Return
`<boolean>` Success
